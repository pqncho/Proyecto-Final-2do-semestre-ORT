# 🚀 ERP Management System

A desktop Enterprise Resource Planning (ERP) application designed to manage employees, managers, departments, budgets, and internal resource allocation.

This project was developed as part of the **Programming II** course at **ORT University Uruguay** and focuses on applying Object-Oriented Programming principles, desktop application development, data persistence, and Artificial Intelligence integration.

---

## ✨ Features

### 👥 Employee Management

* Employee registration and administration
* Curriculum Vitae storage in external text files
* Assignment to departments and managers
* Salary tracking and validation

### 🧑‍💼 Manager Management

* Manager registration and administration
* Employee supervision tracking
* Contact information and seniority management

### 🏢 Department Management

* Department creation and configuration
* Annual budget management
* Employee transfers between departments
* Automatic budget recalculation after transfers

### 💾 Automatic Persistence

* System data is automatically saved
* Ability to start with:

  * An empty system
  * Previously saved data
  * Preloaded sample data

### 📊 Reporting & Analytics

* Department budget utilization report
* Employee distribution by department
* Movement history tracking
* Advanced filtering by month, department, and employee
* CSV export functionality

### 🤖 AI-Powered Insights

* Integration with **Google Gemini**
* Intelligent analysis before employee transfers
* Automatic generation of advantages and disadvantages based on:

  * Employee CV
  * Source department
  * Destination department

### 🎨 Desktop User Interface

* Java Swing graphical interface
* Dynamic components and visual indicators
* Real-time information updates
* Interactive management experience
